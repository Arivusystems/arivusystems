## Quick start

This is a **static HTML/CSS/JS** site.

**Run with auto-refresh (recommended for development):**

```bash
npm install
npm run dev
```

Your browser will open at http://localhost:8000 and **reload automatically** when you save changes.

**Without npm:** use any static server (e.g. `python3 -m http.server 8000`). See **[DOCUMENTATION.md](./DOCUMENTATION.md)** for all options.

## What's inside

- **Pages:** Home (`index.html`), About, Services, Vtiger Solutions, Contact  
- **Stack:** HTML5, CSS/SCSS, jQuery, GSAP (ScrollTrigger, ScrollSmoother), Bootstrap-based layout  
- **Deploy:** Any static host; repo includes `CNAME` for GitHub Pages custom domain

For project structure and local run steps, see **[DOCUMENTATION.md](./DOCUMENTATION.md)**. For **hosting on GitHub Pages** (including custom domain and DNS), see **[GITHUB_PAGES.md](./GITHUB_PAGES.md)**.
